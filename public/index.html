    Bienvenue au Musée Égyptien

    Explorez une collection d’artefacts égyptiens en 3D ! Voici comment naviguer :

    • Déplacement : Utilisez les touches Z (avant), Q (gauche), S (arrière), D (droite).
    • Rotation : Verrouillez la caméra avec C, puis déplacez la souris pour regarder autour (mode FPS). Déverrouillez avec Alt.
    • Zoom : Utilisez la molette pour zoomer sur une œuvre (quand sélectionnée).
    • Rotation autour d’une œuvre : Quand une œuvre est sélectionnée, maintenez le clic gauche et déplacez la souris pour la voir sous différents angles.
    • Interaction : Approchez-vous d’une œuvre et appuyez sur E pour la voir de près. Appuyez sur E ou clic sur "Retour" pour revenir.
    • Aide : Appuyez sur A ou cliquez sur "Musée Égyptien" pour revoir ce guide. Fermez avec "Échap" ou "Commencer l’exploration".
    • Visite guidée : Cliquez sur "Visite Guidée" dans le panneau d’info pour une exploration automatique.
